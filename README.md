# Blessing DevTech Solutions - Premium Digital Agency Website

This is a static portfolio/sample website created to showcase a premium digital agency style using a realistic business direction.

## What's included
- Modern one-page agency website
- Responsive layout
- Dark/light theme toggle
- Scroll reveal animations
- Animated stats counters
- Project filter section
- FAQ accordion
- Contact form UI
- Ready for GitHub Pages

## Files
- `index.html`
- `styles.css`
- `script.js`

## How to use
1. Extract the zip file.
2. Upload the contents to a GitHub repository.
3. Enable GitHub Pages in the repository settings.
4. Set the source to the main branch root.

## Notes
- The contact form is front-end only for now.
- You can later connect it to Formspree, Netlify Forms, EmailJS, or your own backend.
- Update the contact details and pricing to match your final business offer.
